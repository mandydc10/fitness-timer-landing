## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/mandydc10/fitness-timer-landing.git
```

2. Install dependencies:
```bash
cd fitness-timer-landing
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```
